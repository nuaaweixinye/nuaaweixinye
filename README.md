## 🚀 项目经历

### 🏗 编译原理 — 自研编译器
*C · Lex/Yacc · 汇编生成*

从词法分析到目标代码生成，完整实现了类 C 语言的编译器。涵盖词法分析、语法分析、语义分析、中间代码生成、代码优化和目标代码生成等全部阶段。

### 🛩 无人机仿真
*C++ · PX4 · Gazebo · ROS*

基于 PX4 开源飞控和 Gazebo 仿真平台，搭建无人机仿真环境。实现飞行控制算法与路径规划模块，支持多旋翼和固定翼仿真。

### 💪 健身运动 APP
*前后端全栈*

面向健身爱好者的移动端应用，提供训练计划制定、动作指导、运动记录与数据分析等功能。

### 📅 课表管理 APP
*移动端开发*

为大学生打造的智能课表管理工具，支持课程导入、冲突检测、提醒通知和多周课表切换。

### 📚 图书管理系统
*Vue.js · 数据库设计*

完整的图书借阅管理系统，支持图书录入、借阅归还、读者管理、逾期提醒等功能，采用前后端分离架构。

### ✍️ AI 小说创作平台
*Python · NLP · LLM*

基于大语言模型的智能小说创作辅助工具，提供情节生成、角色管理、风格模仿和续写建议等功能。
